# Bank-Loan-Analysis-Excel-SQL
This project analyzes 38,600+ bank loan applications to evaluate loan performance, borrower risk, repayment behavior and revenue impact. Using SQL for analysis and excel dashboards for visualization, the project delivers clear KPIs and business insights used in real-world banking and credit risk teams.

~ High-Impact KPIs

1. Total Loan Applications: 38.6K
2.Total Funded Amount: $435.8M
3. Total Amount Received: $473.1M
4. Average Interest Rate: 12.05%
5. Average Debt-to-Income (DTI): 13.33%

Month-over-Month Growth
  1. Loan Applications: +6.91%
  2. Funded Amount: +13.04%
  3. Amount Received: +15.84%
